# Wander - Listing Platform

## Overview
Wander is a full-stack web application built with Node.js and Express.js for managing listings (like vacation rentals or properties). It includes user authentication, CRUD operations for listings, a review system with ratings, image uploads, and responsive EJS templates powered by Bootstrap.

## Features
- User registration, login, and logout (Passport.js with local strategy)
- Create, read, update, delete listings with title, description, location, price, country, and image
- Add, edit, and delete reviews on listings with star ratings and comments
- Middleware for authentication (`isLoggedIn`), ownership (`isOwner`), and review authorship (`isReviewAuthor`)
- Form validation using Joi schemas
- Flash messages for success/error feedback
- Image upload support (local uploads folder, possible cloud integration)
- Responsive design with Bootstrap 5
- Error handling with custom `ExpressError` and async wrappers
- Database seeding for demo data

## Tech Stack
- **Backend**: Node.js, Express.js
- **Database**: MongoDB, Mongoose ODM
- **Authentication**: Passport.js (passport-local-mongoose)
- **Templating**: EJS
- **Validation**: Joi
- **Sessions**: express-session
- **Utils**: method-override, multer (inferred), connect-flash
- **Frontend**: Bootstrap 5, custom CSS (style.css, rating.css), vanilla JS (script.js)
- **Deployment Ready**: Environment-based MongoDB URI

## Usage
- **Browse listings**: Visit `/listings`
- **View details**: `/listings/:id` (includes reviews, edit/delete if owner)
- **Create listing**: `/listings/new` (requires login)
- **User auth**: `/signup`, `/login`
- **Leave review**: On listing show page (requires login)

## Routes (Key Endpoints)
- `GET /listings` - Index
- `GET /listings/new` - New form
- `POST /listings` - Create
- `GET /listings/:id` - Show
- `GET /listings/:id/edit` - Edit form
- `PUT /listings/:id` - Update
- `DELETE /listings/:id` - Destroy
- `POST /listings/:id/review` - Add review
- `DELETE /listings/:id/review/:reviewId` - Delete review
